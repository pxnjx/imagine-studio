# Imagine Studio - Minimalist Image Editor

A lightweight, browser-based image editing tool designed for speed and privacy. **Imagine Studio** provides essential editing capabilities in a single, portable HTML file without the need for an internet connection or server-side processing.

## ✨ Features

*   **100% Offline:** Works entirely in your browser. Your images are processed locally and never uploaded to any server.
*   **Essential Toolkit:**
    *   **Crop & Resize:** Precision adjustments for any aspect ratio.
    *   **Image Compression:** Reduce file size efficiently for web use.
    *   **Color Adjustments:** Fine-tune brightness, contrast, and saturation.
*   **Minimalist & Fast:** No bloated libraries—just clean, high-performance code.
*   **Portable:** A single-file solution that you can carry on a USB drive or host anywhere.
